# Deep Root

古代の種として目覚め、世界樹の最深部から地上を目指すファンタジー放置RPG。

## プロジェクト概要
- **Version**: 0.3.0
- **Concept**: 『Your Chronicle』の深淵なリソース管理と、現代的なUI/UXの融合。
- **Focus**: Mobile First Design, Glassmorphism, Interactive Micro-animations.
- **Tech Stack**: React, Vite, Zustand, Tailwind CSS (v4)
